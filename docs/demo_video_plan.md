# End-to-End Demo Video Plan

This document outlines the plan for recording the end-to-end demo video of the humanoid robotics book.

## Demo Scenario
- **Goal**: Showcase the voice-commanded autonomous humanoid robot picking up a red cup in simulation.
- **Key Steps**:
    1. Start Isaac Sim and ROS 2 environment.
    2. Load the humanoid robot and a table with a red cup.
    3. User gives voice command: "Pick up the red cup from the table."
    4. Robot processes command (VLA model).
    5. Robot executes action (locomotion, manipulation).
    6. Robot successfully picks up the red cup.
- **Duration**: Target < 90 seconds.

## Recording Logistics
- **Tool**: [e.g., OBS Studio, ScreenFlow]
- **Environment**: Clean Ubuntu 22.04 with all software installed.
- **Narration**: Clear, concise explanation of each step.

## Success Criteria
- Video clearly demonstrates SC-003.
- Video is under 90 seconds.
- Audio is clear and professional.
